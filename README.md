## Folder Structure

- `android-apps` – Android project scratch space and shared utilities
- `lab-01` – Introductory tooling and Git exercises
- `lab-02` – Test-driven development warm-up
- `lab-03` – Continuous integration practice
- `lab-05` – Firestore integration lab
- `lab-06` – UI testing and Espresso practice
- `lab-07` – Advanced JUnit scenarios
- `lab-08` – Capstone Android app lab
